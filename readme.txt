A location listing plugin, built with flexibility in mind.

== Description ==
xLocate is a plugin that allows you to create and manage listings with locations on your WordPress website. xLocate uses the Google API to easily and intuitively add, listing and add them to a Map. Users can search locations, by specifying a  map location and using various filters to refine the search.

It's designed and coded to be easy to use and extendible.

**How to Add in frontend :**

You can add location search to any of the pages, post types in frontend by using shortcodes

* [xlocate_search_form]
* [xlocate_map]

**Find usage of these shortcodes in our [documentation](https://codemanas.com/docs "documentation").**

== Installation ==
Installation Instructions :
1. Unzip xlocate.zip and Upload xlocate folder to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. Add a shortcode in any of page to show listings in frontend.
